\f0\fs24 \cf0 - How to Install:\
\
If you have Flash Player installed simply double click on the .swf file to play, if you don't,  use your browser (preferable Chrome or Fire Fox), simply drop the .swf in a new tab to start the game.\
\
ELIGO is made with Flixel, sin this folder is included all the game assets and code needed (Flixel included with Flixel Power Tools) to compile and make your own build of the game. Simply make a new project in Flash Builder add the files and build.\
\
- Concept: \
\
ELIGO means "I choose" in latin. \
\
In a regular-repetititve-short period of time, resembling the sound of a heart beat, you have to make a binaural choice (or maybe do nothing), following your interest in life\'85 following your heart.\
\
The smallest decisions change the course of your lifetime.\
\
- How to play:\
\
To start the game click on the screen. To choose you can either click on the image or use the LEFT and RIGHT arrow keys, or you can simply do nothing.\
